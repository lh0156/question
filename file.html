파일 디렉토리 조작

요구사항.001

특정 파일의 경로를 입력받아 정보를 출력하시오.

입력

파일 경로 : D:\class\java\file\test.txt

출력

파일명 : test.txt 
종류 : txt 파일 
파일 크기 : 45B

조건

  • 파일 크기 단위 변환 출력(소수이하 1자리까지) : B, KB, MB, GB, TB

요구사항.002

특정 폴더의 특정 파일(확장자)만을 출력하시오.

입력

폴더 선택 : D:\class\java\editplus
파일 확장자 : java

출력

Ex01_Basic.java
Ex02.java
Ex03_Variable.JAVA
..
Ex45_Random.java

입력

폴더 선택 : D:\class\java\editplus
파일 확장자 : txt

출력

필기.txt
다시 정리할 것.TXT
북마크.txt
..

조건

  • 확장자 대소문자 구분없이

요구사항.003

특정 파일을 이동하고, 중복되는 경우를 처리하시오.

입력

파일 이동을 실행합니다.
같은 이름을 가지는 파일이 이미 존재합니다. 덮어쓸까요?(y/n) y

출력

y. 파일을 덮어썼습니다.
n. 작업을 취소합니다.

조건

  • D:\class\java\file\AAA\test.txt → D:\class\java\file\BBB\test.txt 로 이동하는 것이 목적
  • BBB 폴더에 이미 test.txt가 있을 경우 덮어쓰거나(y) 작업을 취소(n) 하시오.

요구사항.005

리소스

  • 확장자별 카운트
  • 파일 다운로드

요구사항.007

'MusicA와 MusicB 폴더를 서로 비교해서 양쪽 폴더에 모두 존재하는 파일만을 출력하시오.(중복 파일 알아내기)

  • 파일 다운로드

출력

BAAM - 모모랜드 (MOMOLAND).mp3 
Dejavu - 볼빨간사춘기.mp3 
SoulMate (Feat. 아이유) - 지코 (ZICO).mp3

  • MusicA에만 존재하는 파일을 출력하시오.

요구사항.008

'delete' 폴더를 삭제하시오.

리소스

  • 폴더 삭제 > delete
  • 파일 다운로드

출력

폴더를 삭제했습니다. 
삭제된 폴더는 3개이고, 파일은 6개입니다.

도움

  • 재귀 메소드

요구사항.009

폴더의 위치와 상관없이 모든 폴더 내의 파일들을 찾아 한번에 크기를 비교하고, 크기가 큰 순으로 정렬하시오.

리소스

  • 크기 정렬
  • 파일 다운로드

출력

[파일명]    [크기]   [파일이 들어있는 폴더]
aaa.exe    57KB    AAA 
bbb.exe    50KB    BBB 
ccc.dat    41KB    AAA 
ddd.txt    20KB    CCC 
..

도움

  • 버블 정렬 알고리즘 검색

요구사항.010

각 파일들을 아래의 조건에 따라 폴더를 나누어 정렬(이동)하시오.

리소스

  • 직원
  • 파일 다운로드

출력

분류가 완료되었습니다.

조건

  • 1차 : 직원 이름으로 폴더 생성
  • 2차 : 년도별로 폴더 생성
  • 3차 : 각 파일을 직원 > 년도 폴더에 이동
