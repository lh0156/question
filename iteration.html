요구사항.014

최대 9자리 정수를 입력받아 각자리의 홀수 숫자합과 짝수 숫자합을 구하시오.

입력

9자리 숫자 입력 : 273645281

출력

짝수의 합 : 22
홀수의 합 : 16

조건

  • 9자리가 넘어가면 동작 금지
  • 9자리 이하는 동작 가능

도움

  • Math.pow() 메소드 사용(제곱값 구하는 메소드)
  • Math.pow(10, 2) → 100
  • Math.pow(10, 3) → 1000
  • Math.pow(10, 4) → 10000
