예외처리

요구사항.001

예외처리를 하시오.

조건

  • 숫자를 입력받아 합을 구하고 연산 과정을 출력하시오.
    • 숫자 입력 범위: 1~100
  • 사용자 입력 횟수는 N회 입력하시오.
  • 사용자가 '0'을 입력하면 연산을 중단하시오.
  • 여러가지 예외처리를 한 후 예외 발생 횟수를 출력하시오.
  • 예외 종류
    • 숫자가 아닌 데이터 입력 횟수 > NumberFormatException
    • 음수 입력 횟수 > 사용자 예외(Exception)
    • 범위 초과 횟수 > 사용자 예외(Exception)
    • 아무것도 입력안한 횟수 > NumberFormatException

입력

숫자입력: 5
숫자입력: 2
숫자입력: hello
숫자입력: 43
숫자입력: 11
숫자입력:
숫자입력: 9
숫자입력: -3
숫자입력: 123
숫자입력: 20
숫자입력: 111
숫자입력: 0

출력

[연산결과]
5 + 2 + 43 + 11 + 9 + 20 = 90

[예외처리결과]
숫자아님: 1회
음수입력: 1회
범위초과: 2회
입력안함: 1회
